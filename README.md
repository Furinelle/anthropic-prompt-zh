# CL4R1T4S

CL4R1T4S is an educational archive of Chinese translations and source materials for publicly circulated Anthropic model prompts. The project is intended for prompt engineering study, model-behavior research, and comparative analysis of system prompt design.

## Purpose

This repository focuses on making prompt materials easier to read and compare for Chinese-speaking researchers and practitioners. It can be used to study instruction hierarchy, safety policy wording, tool-use protocols, formatting constraints, refusal behavior, memory design, and other prompt engineering patterns.

## Contents

- `ANTHROPIC/` contains Anthropic-related prompt materials and notes.
- Files ending in `.zh.md` are Simplified Chinese translations.
- Original filenames are preserved where possible so translated documents can be compared with their source files.

## Notes

This is an unofficial research and learning repository. The materials are provided for reference, translation, and analysis only. The repository is not affiliated with Anthropic, and it should not be treated as product documentation or an authoritative source for current Anthropic behavior.

Some prompt materials may be outdated, speculative, or sourced from public leaks and community circulation. Use them critically, verify current product behavior from official documentation when accuracy matters, and respect applicable laws, platform terms, and intellectual property rights.
